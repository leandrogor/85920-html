Ejercicio análisis de un documento html con las herramientas del navegador Chrome.

Analice la página web del sitio www.frc.utn.edu.ar con Chrome devtools.
Responda las siguientes preguntas:

1. ¿Cuantos Kilobytes se transfieren al navegador para visualizar la página de manera completa? Cuanto tiempo demora en cargar la página. Si se utiliza el caché del navegador, demora lo mismo?

Se transfieren 1,7 MB. Demora 2,40 segundos. Sin cache demora más tiempo.

2. ¿La página posee errores o warnings?

Si, debido a que tengo un bloqueador de anuncios en ejecución.

3. ¿Cuál es el archivo que demora mas tiempo en descargarse de la página?

Tarda en descargarse más el archivo cuyo nombre es "www.frc.utn.edu.ar" de tipo documento.

4. ¿Todos los archivos provienen del mismo servidor?

No, algunos archivos vienen de otros servidores como YouTube. Se encuentran en la pestaña source.

5. ¿Cuántas imágenes posee?, qué tiempo que demora en descargarlas? ¿Cual es el imagen de mayor tamaño en kilobytes que se descarga del sitio?

Posee 69 imágenes. Demora 3,83 segundos. La imagen de mayor tamaño es una del logo de YouTube (166 kB).

6. ¿Cuántos archivos de hojas de estilo utiliza?

Utiliza 4 archivos css.

7. En qué versión de html indica el documento que está realizada la página?

Utiliza la versión de HTML 1.0

8. ¿Los archivos javascript y/o css están minificados?

Los archivos si estan minificados. Se puede ver en la parte inferior izquierda de la ventana sources {} cuyo archivos esten minificados.

9. ¿Qué tiempo demoraría la página en descargar en un equipo con señal de datos de una red Regular 3G?

La pagina tarda en descargar 11.14 segundos con una señal de datos 3G Fast.

10. La página es compatible con el estándar html indicado en la cabecera? Analice la misma en https://validator.w3.org/.

No, no es compatible con el estandar XHTML. Posee 23 errores.

11. Analice las recomendaciones para mejorar la velocidad de carga de la página en https://developers.google.com/speed/pagespeed/.

Respuesta 11.
